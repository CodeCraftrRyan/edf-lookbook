Create an EDF Lookbook

Simply upload a CSV file and click Upload. A pdf will auotmatically be downloaded to your desktop!

To create a lookbook your CSV file must have any or all of the following column names:

  • headshot Please note, headshot images are taken directly from Salesforce and should be formated as first_last1.jpg
  • name
  • company
  • title
  • additional
  • description
